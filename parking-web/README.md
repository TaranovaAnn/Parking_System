# Parking Web

Фронтенд для системы контроля доступа на парковку.

Основная инструкция запуска и проверки находится в корневом `README.md`.

Коротко:

```bash
cd parking-web
npm install
npm start
```

Frontend по умолчанию:

`http://127.0.0.1:4200`

Если порт занят:

```bash
npx ng serve --host 127.0.0.1 --port 4300
```
